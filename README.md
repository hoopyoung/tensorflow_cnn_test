# tensorflow_cnn_test

一个用卷积神经网络识别验证码的样例。

## 说明

本案例跟其他案例基本类似，连设计的模型都一样，唯一的区别在于本案例使用的验证码非代码自己生成。

## 验证码来源

来源于网上：https://blog.csdn.net/huplion/article/details/78985186，

文末的有下载链接，大约8W张样本

当然你也可以用验证码生成库去生成

## 模型

5层，前三层卷积，后两层为全连接

原始图像 -- 大小处理 -- 转成灰度图

  输入 (176 * 56 * 1）                   

    卷积（5 * 5 *32）+池化（[1,2,2，1]）,得到（ 88 * 28 * 32）
    
    卷积（5 * 5 *64）+池化（[1,2,2，1]）,得到（ 44 * 14 * 64）
    
    卷积（5 * 5 *64）+池化（[1,2,2，1]）,得到（ 22 * 7 * 64）
    
    全连接  1024
    
    全连接  248
    
    


## 使用说明

```
$ pip3 install -r requirements.txt

# 训练模型，如太慢或者没有自己的GPU，可略过，使用已经训练好的模型

```

本案例只用于表明CNN识别网站英文数字验证码的一个过程，由于使用的是某网站实际使用的验证码，请勿将该识别用于目标网站，后果自负。
